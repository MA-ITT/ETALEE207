### We need your help in translating Yukki Music Bot. How to Contribute?

You can edit [`ar.yml`](https://github.com/TeamYukki/public/blob/master/strings/langs/ar.yml) present in langs folder to your own language and send us the edited file at [@klaerSupport](https://t.me/sasa_body1)
